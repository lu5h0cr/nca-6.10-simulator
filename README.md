# NCA 6.10 Private Practice Simulator

Simulador privado para GitHub Pages con requisito de aprobación de 80%.

## Estructura

```txt
/
├── index.html
└── data/
    └── questions.js
```

## Cómo cargar preguntas

Editar `data/questions.js`.

`correctAnswerIndex` empieza en 0:

```txt
A = 0
B = 1
C = 2
D = 3
```

## Importante

No publiques material comprado o no autorizado en un repositorio público.
Para material sensible, usá un repositorio privado.
