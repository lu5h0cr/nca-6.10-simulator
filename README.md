# NCA 6.10 - Simulador local con imagenes

Estructura:

/
├── index.html
├── data/
│   └── questions.js
├── images/
│   └── qxxx_exhibit.png
└── audit/
    ├── image_audit.csv
    └── questions_audit_with_images.csv

## Uso

Abrir index.html localmente.

## Notas

- El simulador lee data/questions.js.
- Las preguntas con imagen usan el campo image: "./images/archivo.png".
- No borrar la carpeta images si hay preguntas con exhibit.
